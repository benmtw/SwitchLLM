## Logout claude code

when switching models i have read it is helpful to logout of claude code before switching models

Currently the code prints a warning to do this manually, it may be able to do this via claude command line usage, eg claude -p "/logout"

Note reference here https://www.kimi.com/code/docs/en/more/third-party-agents.html#claude-code about setting hasCompletedOnboarding flag (see script they provide) to stop it trying to log you in after having done this
